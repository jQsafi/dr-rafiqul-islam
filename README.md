# ড. এস. এম. রফিকুল ইসলাম বাচ্চু - অফিশিয়াল বায়োগ্রাফি

গাজীপুর-৩ আসনের সংসদ সদস্য ড. এস. এম. রফিকুল ইসলাম বাচ্চুর একটি প্রিমিয়াম এবং রেসপনসিভ বায়োগ্রাফি ওয়েবসাইট।

## 🌟 বৈশিষ্ট্যসমূহ (Features)
- **প্রিমিয়াম ডিজাইন:** আধুনিক গ্লাস-মরফিজম (Glassmorphism) এবং ডার্ক থিম।
- **রেসপনসিভ লেআউট:** ডেস্কটপ, ট্যাবলেট এবং মোবাইলে স্বাচ্ছন্দ্যে ব্যবহারযোগ্য।
- **ইন্টারঅ্যাক্টিভ টাইমলাইন:** রাজনৈতিক জীবনের গুরুত্বপূর্ণ মাইলফলকগুলোর চিত্রায়ন।
- **উন্নয়নমূলক কর্মকাণ্ড গ্যালারি:** নির্বাচনী এলাকার প্রধান প্রকল্পগুলোর প্রদর্শনী।
- **সোশ্যাল মিডিয়া ইন্টিগ্রেশন:** ফেসবুক, টুইটার, ইউটিউব এবং হোয়াটসঅ্যাপের সরাসরি লিঙ্ক।
- **স্মুথ অ্যানিমেশন:** স্ক্রল-রিভিল এবং মাইক্রো-ট্রানজিশন ইফেক্ট।

## 🛠️ প্রযুক্তি (Tech Stack)
- **HTML5:** আধুনিক এবং সেমান্টিক কাঠামো।
- **CSS3:** ভ্যানিলা সিএসএস (Custom Properties, Flexbox, Grid)।
- **JavaScript:** ইন্টারঅ্যাক্টিভিটি এবং অ্যানিমেশনের জন্য ভ্যানিলা জেএস।
- **Fonts:** Google Fonts (Noto Sans Bengali)।
- **Icons:** Font Awesome।

## 🚀 হাউ টু রান (How to Run)
১. এই রিপোজিটরি ক্লোন করুন: `git clone https://github.com/jQsafi/dr-rafiqul-islam.git`
২. `index.html` ফাইলটি যেকোনো মডার্ন ব্রাউজারে ওপেন করুন।

## 📁 প্রজেক্ট স্ট্রাকচার
- `index.html`: মূল ওয়েবসাইট স্ট্রাকচার।
- `style.css`: সকল ডিজাইন এবং স্টাইলিং।
- `script.js`: অ্যানিমেশন এবং লজিক।
- `assets/`: ছবি এবং গ্রাফিক্স ফাইল।

## 🛡️ ডেপ্লয়মেন্ট (Deployment)
এই প্রজেক্টে অটোমেটিক ডেপ্লয়মেন্ট সেটআপ করা আছে।

### **১. মেইন ব্রাঞ্চ (main):**
- আপনি সরাসরি `main` ব্রাঞ্চে পুশ করতে পারেন অথবা Pull Request (PR) ব্যবহার করতে পারেন। 
- যেকোনো পরিবর্তনের পর স্বয়ংক্রিয়ভাবে ওয়েবসাইট আপডেট হয়ে যাবে।

### **২. অটো-ডেপ্লয়মেন্ট:**
এখন থেকে আপনার কম্পিউটারে আর `npm run deploy` করার প্রয়োজন নেই। আপনি যখনই `main` ব্রাঞ্চে কোনো কোড পুশ করবেন, **GitHub Actions** নিজে থেকেই সাইটটি আপডেট করে দেবে।
